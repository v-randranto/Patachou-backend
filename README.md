# Frameworks

Node/Express, MongoDB/Mongoose

## Variables d'environnement

Les variables suivantes sont paramétrées dans le fichier .env :

Pour l'envoi d'email:
    EMAIL_SERVICE=""
    EMAIL_USER=""
    EMAIL_PASS=""
    EMAIL_FROM=""

des options de la session:
    SESSION_NAME=""
    SESSION_SECRET=""
    SESSION_TTL=""

## Fichier .json de configurtion

Pour la BDD mongoDB:
    "db_url"
